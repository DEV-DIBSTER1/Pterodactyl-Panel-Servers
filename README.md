# Pterodactyl-Panel-Servers
Something random. Shows all Pterodactyl servers of a user. Nothing special.

# Installation
1. `npm i`
2. Change `example.config.json` to `config.json`
3. Fill out the configuration file with all the information.
4. `node .`
